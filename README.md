# Academic Research Paper Skill

A Codex skill for planning, drafting, revising, and evaluating academic research papers, research proposals, literature reviews, abstracts, methods sections, and English academic phrasing.

## Install

Copy or clone this repository as:

```text
~/.codex/skills/academic-research-paper
```

On Windows, the path is usually:

```text
C:\Users\<your-username>\.codex\skills\academic-research-paper
```

## Use

Ask Codex to use:

```text
$academic-research-paper
```

Example prompts:

```text
Use $academic-research-paper to help me outline a research proposal.
Use $academic-research-paper to revise my literature review.
Use $academic-research-paper to polish this paragraph in English academic style.
```

## Sources

This skill uses summarized, workflow-oriented guidance adapted from:

- Sacred Heart University Library, "Organizing Academic Research Papers": https://library.sacredheart.edu/c.php?g=29803
- The University of Manchester, "Academic Phrasebank": https://www.phrasebank.manchester.ac.uk/

The Academic Phrasebank is the intellectual property of the University of Manchester. This repository does not copy the Phrasebank phrase lists; it uses a compact functional guide so Codex can generate original, context-specific academic wording.
